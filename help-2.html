Help - YouTube ReVanced

Google has moved to split APKs format since the YouTube v14.41.52 update. The Android package installer still does not support split APKs, so use the Community ReVanced Manager for installing ReVanced apps.

The instructions to use the Community ReVanced Manager are below:

Installation Instrucions:

For the MIUI, follow this separate guide using this link.

For Non-Root:

If your Android device is Non-Rooted, follow these instructions to install the app.

  1. Download and install the latest Community ReVanced Manager from our website.
  2. Open the Community ReVanced Manager app and tap on the blue arrow at the lower right corner.
  3. Install ReVanced microG. (If you want to sign in to your Google account, this app is required to install)
  4. Next, install YouTube ReVanced or YouTube Music ReVanced as your desire, then log in to your Google account and enjoy!
For Root:

If your Android device is Rooted, follow these instructions to install the app.

  1. Download and install the latest Community ReVanced Manager from our website.​
  2. Open the Community ReVanced Manager app and tap on the "Magisk" icon to grant root permissions.​
  3. Once you are done, install YouTube ReVanced or YouTube Music ReVanced.

Important Information:

  1. Disabling "APK signature verification" is no longer required.​
  2. The magisk root version was discontinued and removed from the magisk repository because Google switched to Split APKs format. So we replaced our root installation with the new Community ReVanced Manager, directly mounts revanced on the top of /data/app directory of YouTube, making more easier and faster.​
  3. ReVanced won't block ads while using the casting feature, it doesn't cast the video from your device to the Chromecast/TV. Instead, your android device will act as a remote controller for the YouTube app in your Chromecast/TV. That is why the ads are displayed.

Known Bugs:

  1. Buying items in the non-root version causes the app to crash. Likely due to MicroG but to be honest, it's better this way for your security. Please use the original youtube app to buy the stuff you want.​
  2. If videos are lagging or YouTube ReVanced crashing while watching live videos, enable VP9 codec from codec override from ReVanced Settings. Google forces the ExoPlayer v2, which is still experimental and not stable on unsupported devices. ExoPlayer V2 might crash when return from background playback on some devices.​
  3. Casting in the v16.16.38 won't work because of an upstream microG bug.
  4. Preferred quality options are not working in portrait/vertical videos.

Official Installer - (Ad)ReVanced Options

Using the ytva settings.sh

  1. ​SEARCHFOLDER - Path to search for iYTBP Apks (must be a subfolder on sdcard!)
  2. ​MAGINSTALL - Use magisk module installation instead of root installation.
  3. DETACH - Disconnect You Tube from google play to disable update notifications.
  4. ATTACH - Connect You Tube to google play if previously disconnected.
  5. UNINSTALL - Fully uninstall iYTBP: root version, magisk module, and detach module.

Using the ytva plz files (use only the ones you want and should be placed in /sdcard/Download/)

  1. ​ytva_install_via_magisk.plz - Use magisk module installation instead of root installation
  2. ytva_detach_from_playstore.plz - Detach You Tube from google play to disable update notifications.​
  3. ytva_attach_to_playstore.plz - Attach You Tube to google play if it was previously detached.
  4. ytva_uninstall.plz - Completely uninstall iYTBP: root version, magisk module, and detach module.
