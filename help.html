Help - YouTube ReVanced

Installation Instrucions:

For Non-Root:

If you are using a Non-Rooted Android device, follow these instructions to install YouTube ReVanced:

  1. Remove all previous installations of MicroG and YouTube ReVanced
  2. Download and install the latest version of MicroG here, note that MicroG is a background service so it may require some background permissions.
  3. Download and install the latest version of YouTube ReVanced here.
  4. Enjoy ad-free YouTube viewing.
For Root:

To install the app on your Rooted Android device, follow these simple steps:

  1. Download and install the latest Community ReVanced Manager from our website. â€‹
  2. Open the Community ReVanced Manager app and tap on the "Magisk" icon to grant root permissions.
  3. Now, you can easily install YouTube ReVanced or YouTube Music ReVanced.

How can you build your own custom APK file?

For PC:
  1. Download the latest version of ReVanced Builder from our website:
    • For Windows users, download here.
    • For Linux users, download here.
    • For MacOS users, download here.
    • You can also follow the official author's page here to download.
  2. Run the downloaded file. It will take some time to start and check information. Then, access http://localhost:8000 to enter the builder interface.
  3. Click "Start Paching".
  4. Wait for the builder to download the latest required dependencies, then click "Continue".
  5. Select the application you want to patch. In this case, we choose YouTube.
  6. Select or deselect the patches you want to apply to the original APK of YouTube. You can also keep the default, then click "Continue".
  7. Wait for the builder to finish running, and you will see the newly created APK file in the \revanced folder. You now have your own clean APK file.
  8. Copy it to your phone and install it.
For Android Device:
  1. Download the latest version of Community ReVanced Manager
  2. Install Community ReVanced Manager on your Android device.
  3. Open Community ReVanced Manager and tap on the "Patcher" tab.
  4. Select "YouTube" from the list of apps, or download and select YouTube APK
  5. Select Paches that you want to install.
  6. Tap on the "Build" button.
  7. Community ReVanced Manager will download the necessary files and build the ReVanced YouTube app.
  8. Once the app is built, tap on the "Install" button to install it on your device.
  9. Open the ReVanced YouTube app and enjoy your ad-free, background playback, and PiP experience.
    Here are some additional things to keep in mind:
  • You need to have the latest version of the YouTube app installed on your device in order to build ReVanced YouTube.
  • You also need to have the latest version of MicroG installed on your device in order to sign in to ReVanced YouTube.
  • If you encounter any problems, you can try clearing the cache and data for Community ReVanced Manager and ReVanced YouTube.
