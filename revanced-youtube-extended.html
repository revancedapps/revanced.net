ReVanced YouTube Extended Features

  • Adds splash animation, which was removed in YT v18.19.36+. This patch cannot be used with 'custom-branding-icon' patch
  • Bypass ambient mode restrictions in battery saver mode. This will allow you to continue watching videos in the background even when your phone is in battery saver mode.
  • Change home page to subscription feed. This will make the subscription feed the default page when you open the YouTube app.
  • Rename the YouTube app to the name specified in options.json. This is useful if you want to give the YouTube app a custom name, such as "My YouTube" or "YouTube Pro".
  • Changes the YouTube launcher icon to MMT. This will change the icon of the YouTube app to the MMT logo.
  • Changes the YouTube launcher icon to Revancify Blue. This will change the icon of the YouTube app to the Revancify Blue logo.
  • Changes the YouTube launcher icon to Revancify Red. This will change the icon of the YouTube app to the Revancify Red logo.
  • Add 'double-tap to seek' value. This will allow you to seek through videos by double-tapping on the seekbar.
  • Specifies the package name for YouTube and YT Music in the MicroG build. This is useful if you want to use the MicroG build of YouTube with a different package name, such as "com.google.youtube.vanced" or "com.google.youtube.music.vanced".
  • Adds more playback speed options. This will allow you to watch videos at a variety of playback speeds, from 0.25x to 2.0x.
  • Change seekbar color in video player and video thumbnails. This is useful if you want to change the color of the seekbar in the video player and video thumbnails.
  • Adds ability to set default playback speed settings. This will allow you to set a default playback speed for all videos.
  • Adds ability to set default video quality settings. This will allow you to set a default video quality for all videos.
  • Disable CronetEngine's QUIC protocol. This is useful if you are having problems with videos not loading or buffering.
  • Disables playing YouTube Shorts when launching YouTube. This will prevent YouTube from automatically playing YouTube Shorts when you launch the app.
  • Disables forced auto captions. This will prevent YouTube from automatically turning on captions for all videos.
  • Disable haptic feedback when swiping. This will disable the haptic feedback that is triggered when you swipe through videos.
  • Disable HDR video. This will disable HDR video playback.
  • Disable landscape mode when entering fullscreen. This will prevent the video from rotating to landscape mode when you enter fullscreen.
  • Disable pip notification when you first launch pip mode. This will prevent the pip notification from appearing when you first launch pip mode.
  • Enables compact control overlay. This will make the control overlay in the video player smaller.
  • Adds debugging options. This will add a number of debugging options to the YouTube app, such as the ability to see the network traffic and the ability to disable ads.
  • Open url outside the app in an external browser. This will open any links that you tap on in the YouTube app in an external browser, instead of opening them in the YouTube app.
  • Enables minimized and background playback. This will allow you to minimize or play videos in the background even when you are using other apps
